Rules
A) General Rules
  1. This is a 2-4 player game: Red vs Blue vs Yellow vs Green.
  2. Each player controls 5 soldier units, each with a unique identity (ID 1-5).
  3. The objective is to move soldiers from your Start Line to the Finish Line and score more points than your opponent.
  4. The game has three phases: Placement, Main, and Game Over.
  5. The first player is chosen randomly.
  6. Each player has three holding areas: Inventory, Dead, and Finished.
  7. At game start, all 5 soldiers begin in Inventory.
  8. During placement, players alternate placing one soldier on their own Start Line until all soldiers are placed.
  9. Soldiers eliminated in Main Phase move to Dead.
  10. Soldiers that reach Finish move to Finished and are no longer active.
  11. After placement, the game enters Main Phase.
  12. Players alternate turns until the game ends.
  13. Enemy soldier IDs are hidden. An ID is revealed when that soldier uses power, and stays revealed until it dies or finishes.
  14. On each turn, a player chooses one action: Roll Dice or Use Power (exception: ID 4).
  15. The game ends immediately if only one team remains onboard.
B) Movement Rules
  1. The battlefield is a 12x12 grid.
  2. Movement uses a D3 roll.
  3. After rolling, exactly one soldier must move exactly the rolled value (no partial movement).
  4. If no legal move exists for that roll, the turn is skipped.
  5. To finish, a soldier must land exactly on Finish Line.
  6. A soldier cannot land on a friendly unit.
  7. If a soldier lands on an enemy unit, the stationary enemy is eliminated.
  8. After rolling, player cannot switch to power that turn.
C) Power Rules
  1. Instead of rolling, a player may use a soldier power.
  2. Powers by ID:
    • ID 1 Recruit: no special power.
    • ID 2 Blockader: places barricades.
    • ID 3 Sniper: straight-line elimination in chosen direction.
    • ID 4 Freeze: affects 8 adjacent squares around itself; frozen soldiers cannot move or use powers.
    • ID 5 Nuke: similar to sniper but across three parallel battle lines in chosen orientation.
  3. A power can be used once per lifetime.
  4. Resurrected soldiers regain their power.
  5. Using power reveals that soldier’s identity.
  6. Exception: ID 4 may use freeze after moving in the same turn.
  7. Special interactions (also shown in gameplay):
    • Barricades have placement restrictions.
    • Auto-barricade can trigger when ID 2 is hit by gun/nuke.
    • ID 4 may jump over frozen soldiers.
    • ID 2 may jump over barricades.
    • Power can only be used in battlefield not from start line
  8. If a player has only one soldier left (on board or Start Line), that soldier ignores freeze and barricade effects.
  9. When a soldier leaves board (finished/eliminated) or gets frozen, its persistent effects(barricades/freezes) are removed.
D) Finishing, Resurrection & Scoring
  1. When a soldier reaches Finish, it moves to Finished and is no longer active.
  2. Player gains points equal to that soldier’s ID.
  3. For each soldier that finishes, player must resurrect one soldier from Dead if available.
  4. Resurrected soldiers are placed on Start Line.
E) Game Over & Winning
  1. Game ends immediately if only one team left onboard.
  2. Final scores are compared.
  3. Higher score wins.
  4. A team finishing all its soldier 1st will be awarded +4 bonus points.
  5. +3 bonus for 2nd and +2 bonus for third finishing team.
