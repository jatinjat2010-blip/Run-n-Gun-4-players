Run and Gun is a turn-based tactical board game built around hidden identities, bluffing, and precise positioning.

Each side controls five soldiers with unique IDs (1–5). Your objective is to move units from Start to Finish while outscoring your opponent. The catch: enemy identities stay hidden until they use powers, so every move can be deception.

How it plays:

Place your team on the start line
On each turn, choose one action: roll to move or use a power
Eliminate enemies through movement or ranged abilities
Reach Finish to score points and trigger resurrection opportunities
Win with the highest score when the game ends
Unique powers:

ID 2: Barricade control
ID 3: Sniper line attack
ID 4: Freeze nearby units
ID 5: Nuke-style multi-line attack
